CSV-JavaScript

Conexion de informacion con archivos CSV a Javascript. Se puede visualizar el contenido en Consola o con DOM. Creacion de objetos. Manejo de Arrays con objetos.

Reproduccion Audio con JS

Reproduccion de Audio con JS Los audios estan en un array y se pueden habilitar o desabilitar a traves del DOM.

MySQL - Java

Proyecto para conectar una base de datos con MySQL. Se puede realizar funciones de crear, borrar, cambiar y eliminar items. Crea objetos internamente. La configuracion de la base datos tambien posee unas celdas que estas alimentadas por informacion de otras celdas. Tiene dos formularios para visualizar la base de datos. Puede exportar la informacion de la BBDD a CSV.

CSV - Java - XML

Conexion de informacion con archivos CSV a JAVA. Creacion de objetos, clases abstracta, herencias, polimorfismos. Manejo de Arrays con objetos. Se actualizo el archivo con la creacion de archivos XML.

Audio ♪ + JAVA

Proyecto de reproduccion de archivos WAV en JAVA. Incluye opcion de ingreso de monedas simulando un JUKEBOX.

Audio ♪ + Python

Proyecto de reproduccion de archivos de audio en Python. Incluye opcion de ingreso de monedas simulando un JUKEBOX.

Tienda Online (Design)

Proyecto de diseño HTML con CSS, simulando una tienda Online. Existen apartados en esta pagina que suman proyectos anteriores realizados con JS: audio + csv.
